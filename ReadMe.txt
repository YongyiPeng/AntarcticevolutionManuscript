1. HGT analysis
This workflow focuses on identifying horizontal gene transfer within microbial MAGs.
HGT_analysis/Metachip_PI.sh
This script runs MetaCHIP’s PI module on the dereplicated MAGs to: group input genomes at multiple taxonomic levels for HGT detection, prepare single-copy gene based species trees, build a genome-wide BLAST database for similarity comparison. 

HGT_analysis/Metachip_BP.sh
This script runs MetaCHIP’s BP module to: take the BLAST results and phylogenetic information generated by the PI module, perform HGT detection across multiple taxonomic levels, assign predicted HGT direction using Best-Match approach combined with Ranger-DTL2 (donor to recipient).

2. Structure-informed population genetics 
Population_genetics/1_anvi-gen-contigs-database.sh
Generate Anvi’o contigs databases (.db) from fasta files. Can include external gene calls (e.g., Prodigal outputs) and ignore internal stop codons.

Population_genetics/2_anvi-export-gene-calls.sh
Export gene calls from contigs DB for downstream analysis.

Population_genetics/3_anvi_bowtie.sh
Map reads to contigs using Bowtie2 and generate BAM files. Key points: --no-unal keeps only mapped reads; samtools view -F 4 filters unmapped reads; initialize BAM for anvi’o.

Population_genetics/4_anvi-profile.sh
Profile BAM files to generate single-profile DBs for each sample and calculate coverage, SNVs, SCVs, SAAVs, and SVs.

Population_genetics/5_anvi-merge.sh
Merge multiple single-profile DBs into one merged profile DB.

Population_genetics/6_anvi-gen-variability-profile.sh
Extract variant information from merged profile DB.

Population_genetics/7_anvi-sum.sh
Summarize profiles and coverage for each genome or collection.

Population_genetics/8_anvi-pnps.sh
Calculate pN/pS ratios from single-codon variants (SCVs) for genes of interest.

Population_genetics/9_anvi-structure.sh
Generate structure DB from predicted PDB files.

Population_genetics/10_anvi-structure_run.sh
Run InteracDome for ligand-binding predictions. Integrate protein structure information. Export residue info or ligand-binding distances. Visualize variants on structures.

3. Phylogenetic codon-based selection analysis
Codon-based_selection/Hyphy.sh
This script uses HyPhy to perform codon-based evolutionary selection analyses on phylogenetic gene alignments. It applies MEME, FEL, and RELAX to detect episodic selection, pervasive selection, and changes in selection intensity across microbial genes.
